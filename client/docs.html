Homestuck OST API Documentation

Return to tests

An API for getting information on tracks, artists, and albums from the Homestuck OST. Also allows for rating of tracks and artists.

Credit to the contributors of the Homestuck Music Wiki for compiling this information and allowing its usage.

All json responses start with an id that matches the request URL

GET/HEAD requests

getAllTracks

Returns a json object containing all tracks on the OST and their relevant info.

No params

Format: {tracks:[{name: string, artists: [string], album: string, trackArt: string, urls: [string], rating: number}, ...]}

Example: {tracks:[{name: 'track1', artists: ['artist1, artist2'], album: 'album1', trackArt: 'artURL', urls: ['url1', 'url2'], rating: 9}, ...]}


getTrack

Returns a json object containing information for the searched track

Params: Title (string - name of track)

Format: {name: string, artists: [string], album: string, trackArt: string, urls: [string], rating: number}

Example: With request "track1" - {name: 'track1', artists: ['artist1', 'artist2'], album: 'album1', trackArt: 'artURL', urls: ['url1', 'url2'], rating: 9}


getAllByArtist

Returns a json object containing all tracks made by the searched artist

Params: Name (string - name of artist)

Format: {tracks:[{name: string, artists: [string], album: string, trackArt: string, urls: [string], rating: number}, ...]}

Example: With request 'artist4' - {tracks:[{name: 'track7', artists: ['artist4'], album: 'album1', trackArt: 'artURL', urls: ['url1', 'url2'], rating: 7}, ...]}


getAllArtists

Returns a json object containing information on all artists who have contributed to the OST

No params

Format: {artists: [{artistName: string, rating: number}, ... ]}

Example: {artists: [{artistName: 'artist1', rating: 9}, {artistName: 'artist2', rating: 3} ... ]}


getAllAlbums

Returns a json object containing all albums and their relevant information

No params

Format: {albums:[{albumName: string, albumArt: string, urls: [string], tracks [string]}, ...]}

Example: {albums:[{albumName: 'album1', albumArt: 'artURL', urls: ['url1','url2'], tracks ['track1', 'track2', ...]}, ...]}

POST requests

rateTrack

Adds a numeric rating to a track

Params: Name (string - name of track), Rating (number - rating of track from 0-10)

Supported formats: application/json, application/x-www-form-urlencoded

Examples

Body: {name: 'track1', rating: 9}

Result: {name: 'track1', artists: [...], album: '...', trackArt: '...', urls: [...], rating: 9}

Body: {name: 'track1'}

Result: 400 Error (missing params)

Body: {name: 'notReal', rating: 10}

Result: 404 Error (track not found)


rateArtist

Adds a numeric rating to an artist

Params: Name (string - name of artist), Rating (number - rating of artist from 0-10)

Supported formats: application/json, application/x-www-form-urlencoded

Examples

Body: {name: 'artist1', rating: 7}

Result: {artistName: 'artist1', rating: 7}

Body: {name: 'artist1'}

Result: 400 Error (missing params)

Body: {name: 'notReal', rating: 10}

Result: 404 Error (artist not found)
